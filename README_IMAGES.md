# 이미지 자동 업로드 가이드

## 사용 방법

이미지 폴더에 새 이미지를 추가한 후, 다음 명령어를 실행하면 자동으로 R2에 업로드됩니다:

```bash
npm run upload-images
```

또는

```bash
node scripts/upload-images.js
```

## 동작 방식

1. **로컬 이미지 확인**: `image/` 폴더의 모든 `.png` 파일을 찾습니다
2. **R2 이미지 확인**: R2 버킷에 이미 업로드된 이미지 목록을 가져옵니다
3. **중복 체크**: R2에 없는 이미지만 필터링합니다
4. **자동 업로드**: 중복되지 않은 이미지만 R2에 업로드합니다

## 예시

```
🚀 R2 이미지 자동 업로드 시작...

📁 로컬 이미지: 13개 발견

☁️  R2에 있는 이미지: 9개

📤 업로드할 이미지: 4개

  - diamond_axe.png
  - iron_axe.png
  - mithril_axe.png
  - steel_axe.png

📤 업로드 중: diamond_axe.png...
✅ 업로드 완료: diamond_axe.png

...

📊 업로드 결과:
  ✅ 성공: 4개
```

## 주의사항

- 이미 R2에 있는 이미지는 업로드하지 않습니다 (중복 방지)
- 새 이미지를 추가한 후에만 스크립트를 실행하면 됩니다
- 이미지 파일명은 `.png` 확장자만 지원합니다
